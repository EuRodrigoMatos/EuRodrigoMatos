## 👨‍💻 About Me

I'm a **Full Stack Developer** specialized in building complete systems for **Agribusiness** — from database architecture to modern dark-themed UIs. I turn complex field operations into intuitive digital tools.

- 🌾 Creator of **PantaVisita** — field inspection system for crop monitoring
- 🖥️ Creator of **PantaConect** — corporate ERP with modern dark mode dashboard
- 📡 Built a **network monitoring panel** inspired by Zabbix with dynamic device management
- ⚡ I care about both **clean code** and **exceptional UI/UX**
- 🤝 Open to freelance, remote work, and collaboration

---

## 🚀 Featured Projects

### 🌾 PantaVisita
> Field visit management system for agro-technical inspection

A complete platform for managing technical visits to crops — from scheduling through to PDF report generation. Built with PHP, React and PostgreSQL. Focused on usability for agronomists and farm managers in the field.

`PHP` `React` `TypeScript` `PostgreSQL` `REST API` `PDF Generation`

---

### 🖥️ PantaConect
> Corporate ERP with modern dark mode interface

Full-featured corporate management system with a sleek Dark Mode dashboard. Handles user management, workflow automation, and real-time reporting — designed for operational efficiency in agribusiness companies.

`PHP` `React` `TypeScript` `PostgreSQL` `Dashboard` `Dark UI`

---

### 📡 Network Monitoring Panel
> Zabbix-inspired device monitoring dashboard

Dynamic monitoring panel for infrastructure teams — organizes devices, tracks statuses, and surfaces alerts in real time. Clean and responsive interface for network operations.

`React` `TypeScript` `REST API` `Real-time` `Dashboard`
